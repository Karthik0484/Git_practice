#  GitHub Command Practice Project

##  Project Overview
This project is created to practice essential **Git and GitHub commands** through hands-on implementation.

---

## Commands Practiced

- `git init`
- `git status`
- `git add`
- `git commit`
- `git log`
- `git branch`
- `git checkout`
- `git merge`
- `git remote -v`
- `git push`

---

## Key Learnings

✔ Version control basics  
✔ Staging and committing changes  
✔ Branch creation and switching  
✔ Merging branches  
✔ Connecting local repo to remote repo  
✔ Pushing code to GitHub
